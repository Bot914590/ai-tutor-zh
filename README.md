

## Возможности

- **Грамматика** — проверка предложений, объяснение правил
- **Диалоги** — практика разговорной речи
- **Исправление ошибок** — детальный разбор ошибок
- **Работа локально** — через Ollama, без облачных API
- **GPU-ускорение** — быстрый ответ благодаря видеокарте

## Установка

### Требования

- Python 3.10+
- Ollama с моделью `qwen3.5:4b` (или другой)
- Видеокарта NVIDIA (опционально, для GPU)

### Шаги

```bash
# 1. Клонировать репозиторий
git clone <your-repo-url>
cd ai-tutor

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Установить Ollama (если нет)
# https://ollama.ai

# 4. Скачать модель
ollama pull qwen3.5:4b

# 5. Запустить
python src/agent.py
```

### Команды

| Команда | Описание |
|---------|----------|
| `/grammar` | Режим грамматики |
| `/dialogue` | Режим диалога |
| `/correction` | Исправление ошибок |
| `/help` | Показать справку |
| `/exit` | Выход |

### Пример 

```
[grammar] > 请解释一下"了"的用法
Думаю...
"了" — это частица совершенного вида...
────────────────────────────────────────────────────
```

```yaml

generation:
  temperature: 0.2
  max_tokens: 1024
```
